图

图的定义

图是一种网状非线性结构，图的主要组成是顶点和顶点之间的连线(关联)。

  • V = {x | x ∈ Data}, Data是vertex(顶点)集合

  • VR = {<x,y> | P(x,y) && (x,y ∈ V) }, P(x,y)表示关联属性，<x,y>表示两个顶点之间的弧。

    x是起始点,y是终止点，图分为有向和无向。

基本术语与结论

  • n表示顶点个数，e表示边或弧的数目，(不考虑自环)。

  • 边： 有向完全图有n(n-1)条边，无向完全图有n(n-1)/2条边。

    每个点都与其他n-1有连线，1+2+...+n-1 = (n-1+1) * (n-1)/2为无向图计算。

  • 稀疏图： \(e < nlog_2n\),反之是稠密图。

  • 邻接点：图G = (V, {E}), \((v_1, v_2) ∈ E\)，则两个顶点互为邻接点。或者说弧(边)与顶点相关联。

  • 度：无向图中：顶点v相关联的边的数目叫做度，记作TD；

    有向图中则分为出度和入度，单个顶点v的度为TD = ID + OD，e = \(ΣTD/2\)

  • 权：边上的值，带权的图叫做网或者赋权图。

  • 路径：顶点序列，有向图的路径有方向。路径长度是指边的数目。起始和终点相同称为回路。

    没有其他重复顶点称之为简单路径或者回路。

  • 连通图：无向连通图是任意两点都有路径，有向则是强连通图；

    极大连通子图是子图称为(强)连通向量
